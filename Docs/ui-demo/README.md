# UI Demo Pack

## Objetivo
Demonstrar visualmente a interface do produto antes de iniciar a implementação funcional.

## Arquivos
- `index.html`
- `styles.css`

## Como abrir
Abra [index.html](/home/nikolasa/Downloads/Finanças/Docs/ui-demo/index.html) no navegador.

## O que está demonstrado
- design system base
- boas-vindas e personalização leve
- dashboard mobile
- adicionar gasto
- histórico
- planejamento dark
- cartões
- metas
- integração WhatsApp
- premium
- biblioteca financeira
- simulador e indicadores
- fluxos principais
- estados globais
- desktop companion analítico
- privacidade por padrão e sync opcional com Google

## Uso
Este pacote serve para:
- validar densidade visual
- aprovar hierarquia e linguagem de UI
- decidir se a etapa de produto pode avançar para implementação
